# Doctor AI

FAU CMBB BioTech Hackathon 2026

## Description

We will be collecting patient data through a questionnaire and make suggestions to patients and doctors on based on their health.

## Features

- Gathers lifestyle and health data through APIs
- Analyzes trends in insulin sensitivity
- Sends clear, prioritized suggestions
- Works with clinician dashboards
